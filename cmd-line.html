The Command Line

Long long ago, before you could open files by clicking on them with a mouse...

People used the Command Line Interface to navigate around their computer files

The command line interface (CLI) let users type in text commands for the computer to carry out.

It can still be used today and any programmer worth their salt knows how to get around using it
